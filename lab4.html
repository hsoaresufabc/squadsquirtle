Relatório do Laboratório 4

Introdução

No Laboratório 04, estudamos técnicas de Histogramas e Limiarização. Iniciamos com a teoria, com base nos seguintes documentos:

  • Equalização de Histograma: Tutorial OpenCV
  • Cálculo de Histogramas: Tutorial OpenCV
  • Operações Básicas de Limiarização: Tutorial OpenCV

A seguir, resumimos os principais conceitos de cada tema:


1. Equalização de Histograma

Técnica que melhora o contraste da imagem, redistribuindo os valores de intensidade dos pixels. O tutorial do OpenCV mostra como o método cv::equalizeHist é aplicado, com exemplos em Python, C++ e Java, focando em imagens em escala de cinza.


2. Cálculo de Histogramas

Método essencial para analisar a distribuição de intensidades dos pixels. Com a função cv::calcHist, é possível calcular histogramas de imagens monocromáticas e coloridas, usando máscaras para regiões específicas, oferecendo flexibilidade na análise.


3. Limiarização

Converte imagens em binárias com base em um valor de referência. O OpenCV descreve métodos como limiar simples, adaptativo e Otsu, úteis em tarefas como segmentação, análise de contornos e reconhecimento de formas.

Código 1: Equalização de Histograma em Imagem Estática


Este código realiza a equalização de histograma para melhorar o contraste de uma imagem estática. O processo detalha as seguintes etapas:

      1. Carregamento da Imagem: Usamos imread para carregar a imagem e verificamos se ela foi carregada corretamente.
      2. Conversão para escala de cinza: A função cvtColor converte a imagem colorida para tons de cinza, pois a equalização de histograma é aplicada em imagens monocromáticas.
      3. Equalização do histograma: Com equalizeHist, ajustamos o histograma da imagem para redistribuir as intensidades de pixel e melhorar o contraste.
      4. Exibição dos resultados: Usamos imshow para mostrar tanto a imagem original quanto a equalizada.

##include "opencv2/imgcodecs.hpp"
#include "opencv2/highgui.hpp"

    if (src.empty())
    {
        cout << "Could not open or find the image!" << endl;
        return -1;
    }

    cvtColor(src, src, COLOR_BGR2GRAY);
    Mat dst;
    equalizeHist(src, dst);

    imshow("Source image", src);
    imshow("Equalized Image", dst);

    waitKey();
    return 0;
}

    • Código 2: Equalização de Histograma em Vídeo (Webcam)


    Este código realiza a equalização de histograma para melhorar o contraste de uma imagem estática. O processo detalha as seguintes etapas:

        1. Abrindo a Webcam: Usamos VideoCapture para acessar a webcam e capturar frames.
        2. Conversão e Equalização: Cada frame é convertido para escala de cinza e, em seguida, o histograma é equalizado.
        3. Exibição: Exibe a imagem original e a equalizada em tempo real.
        4. Fechamento: O loop continua até que o usuário pressione uma tecla.

    #include "opencv2/imgcodecs.hpp"
#include "opencv2/highgui.hpp"
#include "opencv2/imgproc.hpp"
#include "opencv2/videoio.hpp"
#include 

using namespace cv;
using namespace std;

int main(int argc, char** argv)
{
    VideoCapture cap(0);
    if (!cap.isOpened())
    {
        cout << "Erro ao abrir a webcam!" << endl;
        return -1;
    }

    Mat frame, grayFrame, equalizedFrame;

    while (true)
    {
        cap >> frame;
        if (frame.empty())
        {
            cout << "Erro ao capturar o frame da webcam!" << endl;
            break;
        }

        cvtColor(frame, grayFrame, COLOR_BGR2GRAY);
        equalizeHist(grayFrame, equalizedFrame);

        imshow("Webcam - Imagem Original em Tons de Cinza", grayFrame);
        imshow("Webcam - Imagem Equalizada", equalizedFrame);

        if (waitKey(30) >= 0)
            break;
    }

    return 0;
}

    O aprendizado adquirido no Laboratório 04 pode ser aplicado em diversas áreas do mundo real, como:

    • Melhoria de Imagens Médicas: Equalização de histograma pode ser usada para realçar imagens de raios-X ou ressonâncias magnéticas, facilitando a visualização de detalhes sutis.

    Essas técnicas são fundamentais em áreas que requerem análise e interpretação de imagens para obter melhores resultados.
