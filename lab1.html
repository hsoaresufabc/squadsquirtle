Relatório do Laboratório 1

Introdução

O Laboratório 1 introduz conceitos fundamentais de visão computacional utilizando a biblioteca OpenCV, que é amplamente empregada em aplicações de processamento de imagens e vídeos. Durante este laboratório, foram explorados métodos básicos para capturar, exibir e gravar imagens e vídeos, estabelecendo as bases para aplicações práticas e avançadas em áreas como segurança, entretenimento e análise visual.

O laboratório foi dividido em três partes principais. A primeira concentrou-se na configuração do ambiente de desenvolvimento, garantindo que todos os recursos necessários estivessem prontos para uso. A segunda parte explorou operações básicas em imagens e vídeos, utilizando exemplos práticos para manipular arquivos visuais. Por fim, a terceira parte focou na gravação de imagens e vídeos, incluindo experimentos com fontes como a webcam e sequências de imagens.

Este laboratório serve como uma introdução prática e técnica, consolidando conhecimentos que serão aplicados em projetos futuros. As atividades desenvolvidas são essenciais para criar familiaridade com a biblioteca OpenCV e entender como utilizá-la para solucionar problemas do mundo real em visão computacional.

Objetivos

O Laboratório 1 tem como principal objetivo introduzir conceitos fundamentais de manipulação de imagens e vídeos utilizando a biblioteca OpenCV. Através de atividades práticas, os participantes aprenderam a configurar o ambiente de desenvolvimento, capturar frames de dispositivos de vídeo e gravar arquivos de saída para análise posterior.

Os objetivos específicos incluem:

  • Entender a estrutura básica da biblioteca OpenCV e seus principais módulos.
  • Implementar programas para leitura e exibição de imagens e vídeos.
  • Explorar técnicas de gravação de vídeos capturados em tempo real.
  • Familiarizar-se com a manipulação de dispositivos de captura, como webcams.
  • Compreender os fundamentos para o processamento visual em projetos futuros.

Configuração do Ambiente

A configuração do ambiente de desenvolvimento é um passo crucial para garantir que os programas possam ser executados corretamente. Neste laboratório, o foco foi na instalação e validação do OpenCV, utilizando ferramentas disponíveis em sistemas baseados em Linux. O processo incluiu os seguintes passos:

  • Instalação do OpenCV por meio do comando sudo apt-get install libopencv-dev, garantindo a disponibilidade das bibliotecas necessárias.
  • Configuração do compilador g++ com suporte à biblioteca OpenCV, utilizando as flags -lopencv_core, -lopencv_highgui e outras relevantes.
  • Criação e compilação de um programa básico para validar a instalação, incluindo a exibição de uma imagem simples para confirmar o funcionamento das bibliotecas.

Além disso, foram utilizados arquivos de instruções específicos fornecidos no Moodle, como o arquivo L1_parte1_instalar_opencv.txt, para guiar os participantes durante a configuração. Essa etapa garantiu que todos os participantes tivessem um ambiente funcional para a execução das atividades do laboratório.

Processamento Básico de Imagens e Vídeos

Na segunda etapa do laboratório, os participantes exploraram operações fundamentais para a manipulação de imagens e vídeos utilizando o OpenCV. O objetivo foi entender as funções básicas que permitem realizar a leitura, exibição e gravação desses dados visuais, servindo como base para aplicações mais avançadas.

Atividades Realizadas

  • Leitura e Exibição de Imagens: A partir do tutorial Getting Started with Images, os participantes desenvolveram um programa para carregar uma imagem (arquivo messi5.jpg), exibi-la em uma janela e salvá-la no formato .png. Essa atividade demonstrou o funcionamento básico das classes cv::Mat e imshow.
  • Leitura e Gravação de Vídeos: Utilizando o tutorial Reading and Writing Videos using OpenCV, os participantes adaptaram um programa para reproduzir o vídeo fornecido (big_buck_bunny.mp4). Além disso, exploraram métodos para gravar vídeos com a classe VideoWriter.
  • Captura de Frames: Foram estudados programas para capturar frames em tempo real da webcam e exibi-los, permitindo interação direta com dispositivos visuais.

Essas atividades introduziram conceitos importantes, como a estrutura de dados cv::Mat, que armazena imagens e vídeos, e as funções de leitura e gravação que facilitam o processamento de arquivos multimídia. Além disso, os participantes compreenderam como manipular a taxa de quadros (FPS) e as dimensões dos vídeos, o que é crucial para otimização em aplicações de visão computacional.

Obtenção de Fotos e Vídeos

A última etapa do laboratório focou na criação e manipulação de imagens e vídeos capturados em tempo real. Os participantes utilizaram programas desenvolvidos para interagir com a webcam, possibilitando a captura de fotos, gravação de vídeos e realização de experimentos práticos.

Atividades Realizadas

  • Captura de Fotos: Cada grupo realizou a captura de uma foto geral com todos os integrantes, utilizando roupas de cores distintas (Roxo, Verde e Azul) para destacar os participantes. As fotos foram ajustadas para melhorar a iluminação, utilizando editores de imagem quando necessário.
  • Foto-Montagem com Avatares: Os integrantes do grupo criaram uma imagem unificada, incorporando avatares personalizados para representar cada participante. Essa atividade promoveu a familiarização com editores de imagem para composição criativa.
  • Gravação de Vídeos: Foram gravados dois vídeos com pessoas e dois vídeos com objetos, simulando diferentes condições:
    • Movimentos lentos para capturar detalhes progressivos.
    • Movimentos rápidos para avaliar a taxa de quadros e resposta do sistema.
    Cada vídeo foi analisado para verificar a qualidade da gravação e a eficiência da captura em tempo real.

Essas atividades consolidaram o uso de dispositivos de captura e reforçaram a importância de ajustar fatores como iluminação e posicionamento para melhorar os resultados. Os códigos utilizados na prática serão discutidos na seção seguinte. Abaixo estão a foto obtida e a montagem com os avatares.

    • Leitura de Vídeos a Partir de Arquivos

    O código video_read_from_file.cpp permite carregar e reproduzir vídeos armazenados localmente. Ele utiliza a classe VideoCapture para abrir o arquivo, verificar sua disponibilidade e reproduzir frame por frame na tela. Além disso, exibe informações sobre a taxa de quadros (fps) e o número total de frames. Esse método é particularmente útil para validar vídeos e realizar análises frame a frame em aplicações como esportes e segurança.

    Exemplo de código:

    // Inicializa o objeto de captura de vídeo
VideoCapture vid_capture("Resources/Cars.mp4");

// Verifica se o vídeo foi aberto com sucesso
if (!vid_capture.isOpened()) {
    cout << "Erro ao abrir o vídeo" << endl;
}

// Obtém informações sobre o vídeo
int fps = vid_capture.get(CAP_PROP_FPS);
int frame_count = vid_capture.get(CAP_PROP_FRAME_COUNT);
cout << "FPS: " << fps << ", Frames: " << frame_count << endl;

// Reproduz os frames
while (vid_capture.isOpened()) {
    Mat frame;
    if (!vid_capture.read(frame)) break;
    imshow("Frame", frame);
    if (waitKey(20) == 'q') break;
}

    Ao executar o arquivo video_read, o vídeo "Cars.mp4" é então carregado e exibido na tela.

    Leitura de Sequências de Imagens

    O código video_read_from_image_sequence.cpp utiliza sequências de imagens para simular um vídeo. Essa técnica é amplamente usada em áreas como efeitos visuais e reconstrução 3D, onde cada frame é tratado como um arquivo de imagem independente.

    Usando o padrão Cars%04d.jpg, o programa carrega cada imagem na sequência e a exibe na tela como frames. Isso permite processar ou editar cada imagem separadamente antes de criar um vídeo final.

    Exemplo de código:

    // Inicializa o objeto de captura de vídeo com a sequência de imagens
VideoCapture vid_capture("Resources/Image_sequence/Cars%04d.jpg");

// Verifica se a sequência foi carregada
if (!vid_capture.isOpened()) {
    cout << "Erro ao abrir a sequência de imagens" << endl;
}

// Reproduz os frames
while (vid_capture.isOpened()) {
    Mat frame;
    if (!vid_capture.read(frame)) break;
    imshow("Frame", frame);
    if (waitKey(20) == 'q') break;
}

    Ao executar o arquivo video_read_from_image_sequence, o vídeo "Cars.mp4" é montado a partir das imagens (frames) e é carregado e exibido na tela.

    Vídeo"Cars.mp4"

    Leitura de Vídeo Diretamente da Webcam

    O código video_read_from_webcam.cpp utiliza a webcam como fonte de entrada para capturar frames em tempo real. Essa funcionalidade é ideal para aplicações interativas, como vigilância ao vivo, videoconferências ou sistemas de reconhecimento em tempo real.

    Ele cria um objeto VideoCapture que acessa a webcam padrão do sistema. Os frames capturados são exibidos na tela, e o programa permanece em execução até que o usuário pressione a tecla q.

    Exemplo de código:

    // Inicializa a captura de vídeo pela webcam (0 indica a câmera padrão)
VideoCapture vid_capture(0);

// Verifica se a webcam está acessível
if (!vid_capture.isOpened()) {
    cout << "Erro ao acessar a webcam" << endl;
}

// Captura e exibe os frames
while (vid_capture.isOpened()) {
    Mat frame;
    if (!vid_capture.read(frame)) break;
    imshow("Frame", frame);
    if (waitKey(20) == 'q') break;
}

    Gravação de Vídeo a Partir da Webcam

    O código video_write_from_webcam.cpp combina a captura de frames da webcam com a funcionalidade de gravação em vídeo. Ele utiliza a classe VideoWriter para salvar os frames capturados em um arquivo de vídeo no formato .avi.

    O programa captura os frames em tempo real, permitindo que o usuário visualize e armazene o vídeo simultaneamente. Essa funcionalidade é útil em cenários como monitoramento em tempo real ou gravação de reuniões e apresentações.

    Exemplo de código:

    // Inicializa a captura de vídeo pela webcam
VideoCapture vid_capture(0);

// Obtém largura e altura dos frames capturados
int frame_width = static_cast<int>(vid_capture.get(CAP_PROP_FRAME_WIDTH));
int frame_height = static_cast<int>(vid_capture.get(CAP_PROP_FRAME_HEIGHT));

// Inicializa o gravador de vídeo
VideoWriter output("output.avi", VideoWriter::fourcc('M', 'J', 'P', 'G'), 20, Size(frame_width, frame_height));

// Captura e grava os frames
while (vid_capture.isOpened()) {
    Mat frame;
    if (!vid_capture.read(frame)) break;
    output.write(frame);
    imshow("Frame", frame);
    if (waitKey(20) == 'q') break;
}

    Ao executar o arquivo video_write_from_webcam, será feita a captura do vídeo da webcam e será gerado um arquivo de vídeo no formato .avi. , porém para fins de compatibilidade com os navegadores, o vídeo foi convertido para o formato .mp4.

    • Gravação de Vídeos de Arquivos

    O código video_write_to_file.cpp permite ler frames de um vídeo existente e gravá-los em um novo arquivo. Ele combina a funcionalidade de leitura da classe VideoCapture com a gravação por meio de VideoWriter, possibilitando salvar modificações feitas nos frames originais.

    Essa abordagem é útil para operações como alteração de resolução, aplicação de filtros ou edição de vídeos para criar novos conteúdos. O exemplo abaixo mostra como isso é feito:

    Exemplo de código:

    // Inicializa a captura de vídeo a partir de um arquivo
VideoCapture vid_capture("Resources/Cars.mp4");

// Obtém largura, altura e inicializa o gravador de vídeo
int frame_width = static_cast<int>(vid_capture.get(CAP_PROP_FRAME_WIDTH));
int frame_height = static_cast<int>(vid_capture.get(CAP_PROP_FRAME_HEIGHT));
VideoWriter output("Resources/output.avi", VideoWriter::fourcc('M', 'J', 'P', 'G'), 20, Size(frame_width, frame_height));

// Lê os frames e grava no novo arquivo
while (vid_capture.isOpened()) {
    Mat frame;
    if (!vid_capture.read(frame)) break;
    output.write(frame);
    imshow("Frame", frame);
    if (waitKey(20) == 'q') break;
}

    Conclusão

    Este laboratório apresentou uma introdução prática e detalhada ao uso do OpenCV para processamento de imagens e vídeos, abordando conceitos fundamentais e fornecendo uma base sólida para o desenvolvimento de aplicações em visão computacional. As atividades realizadas permitiram compreender e aplicar operações essenciais, como leitura, exibição e gravação de dados visuais, além de explorar a interação com dispositivos de captura, como webcams.

    A configuração do ambiente, o estudo de operações básicas e a implementação de programas práticos consolidaram o aprendizado, permitindo aos participantes trabalhar com diferentes formatos de entrada e saída, tanto em imagens estáticas quanto em vídeos. A análise dos vídeos gravados mostrou a importância de fatores como iluminação, posicionamento e taxa de quadros para garantir a qualidade do material produzido.

    Em conclusão, as habilidades adquiridas neste laboratório servirão como referência para projetos futuros, tanto no contexto acadêmico quanto profissional. As técnicas exploradas são essenciais para aplicações modernas, como monitoramento, reconhecimento de padrões e sistemas interativos baseados em visão computacional.
