Relatório do Laboratório 2

Introdução

O Laboratório 2 tem como objetivo explorar as técnicas de filtragem de imagens utilizando a biblioteca OpenCV. A filtragem é uma operação essencial no processamento de imagens, permitindo reduzir ruídos, suavizar detalhes ou destacar elementos importantes para análises futuras.

Durante este laboratório, serão estudadas e implementadas quatro principais técnicas de filtragem: filtro de média, filtro Gaussiano, filtro mediano e filtro bilateral. Cada técnica será aplicada em imagens previamente capturadas no Laboratório 1, com diferentes tamanhos de kernel, para analisar os efeitos das alterações nos parâmetros de filtragem.

A parte experimental inclui a aplicação dessas técnicas em imagens com e sem ruído, a análise dos resultados e o desenvolvimento de programas interativos para capturar imagens da webcam e aplicar os filtros em tempo real.

Este laboratório visa não apenas aprofundar o conhecimento sobre filtros em processamento de imagens, mas também fornecer ferramentas práticas para resolver problemas como redução de ruídos e preparação de imagens para algoritmos mais avançados, como segmentação e reconhecimento.

Objetivos

O objetivo deste laboratório é explorar técnicas de filtragem de imagens, utilizando funções específicas do OpenCV para suavização e redução de ruídos. A filtragem é uma etapa fundamental no processamento de imagens, permitindo melhorar a qualidade visual e preparar os dados para operações posteriores, como segmentação e reconhecimento.

Este laboratório visa atingir os seguintes objetivos específicos:

  • Implementar os filtros de média, Gaussiano, mediano e bilateral em imagens obtidas no Lab 1, utilizando diferentes tamanhos de kernel.
  • Analisar os efeitos da variação do tamanho do kernel na suavização e preservação de detalhes da imagem.
  • Estudar a eficácia dos filtros em imagens com ruído do tipo sal-e-pimenta.
  • Desenvolver programas interativos para capturar imagens em tempo real pela webcam e aplicar os filtros de forma dinâmica.
  • Proporcionar uma compreensão prática das diferenças entre os filtros e suas aplicações em problemas reais.

Parte Experimental

Nesta seção, foram realizados experimentos para entender e aplicar os diferentes tipos de filtragem em imagens capturadas no Laboratório 1. Os filtros utilizados incluem: filtro de média, filtro Gaussiano, filtro mediano e filtro bilateral.

Etapas Realizadas:

  • Aplicação de Filtros: Foram aplicados os filtros mencionados em imagens com diferentes tamanhos de kernel (3x3, 5x5, 7x7 e 11x11), analisando os resultados obtidos e salvando as imagens resultantes em formato .jpg.
  • Adição de Ruído: A mesma filtragem foi repetida após a adição de ruído tipo sal-e-pimenta nas imagens originais. Essa etapa foi importante para avaliar a eficácia de cada técnica na redução de ruídos.
  • Captura de Imagem em Tempo Real: Um programa foi desenvolvido para capturar imagens da webcam, aplicar dois dos filtros mais relevantes identificados anteriormente e permitir o salvamento da imagem filtrada pressionando a tecla S.
  • Desafio Interativo: Foi implementado um programa avançado que permite ao usuário selecionar o tipo de filtragem e o tamanho do kernel por meio de teclas no teclado, oferecendo maior flexibilidade na análise dos filtros.

    • Cada experimento foi realizado com base no roteiro fornecido e utilizando funções do OpenCV, como blur(), GaussianBlur(), medianBlur() e bilateralFilter(). Os resultados obtidos foram avaliados quanto à suavização, preservação de bordas e eficácia na remoção de ruídos.

    Códigos Utilizados

    Código: Aplicação do Filtro de Média

    #include <opencv2/opencv.hpp>
#include <iostream>

using namespace cv;
using namespace std;

int main() {
    // Carregar a imagem original
    Mat img = imread("input.jpg");
    if (img.empty()) {
        cout << "Erro ao carregar a imagem" << endl;
        return -1;
    }

    // Aplicar o filtro de média com diferentes tamanhos de kernel
    Mat img_blur_3x3, img_blur_5x5, img_blur_7x7;
    blur(img, img_blur_3x3, Size(3, 3));
    blur(img, img_blur_5x5, Size(5, 5));
    blur(img, img_blur_7x7, Size(7, 7));

    // Exibir os resultados
    imshow("Original", img);
    imshow("Filtro de Média - 3x3", img_blur_3x3);
    imshow("Filtro de Média - 5x5", img_blur_5x5);
    imshow("Filtro de Média - 7x7", img_blur_7x7);

    // Salvar as imagens resultantes
    imwrite("media_3x3.jpg", img_blur_3x3);
    imwrite("media_5x5.jpg", img_blur_5x5);
    imwrite("media_7x7.jpg", img_blur_7x7);

    waitKey(0);
    return 0;
}

    Análise de Resultados

    A aplicação dos diferentes filtros em imagens com e sem ruído apresentou resultados distintos, reforçando as características específicas de cada técnica de filtragem. Abaixo, analisamos os principais aspectos observados durante os experimentos:

    1. Comparação Entre Filtros

    O filtro de média suavizou as imagens de forma eficiente, mas apresentou perda significativa de detalhes, especialmente em bordas. O filtro Gaussiano mostrou-se superior na preservação de bordas, sendo mais robusto para imagens sem ruído extremo. Já o filtro mediano destacou-se na remoção de ruídos do tipo sal-e-pimenta, preservando detalhes estruturais importantes. Por fim, o filtro bilateral proporcionou uma boa suavização ao mesmo tempo em que preservou bordas e detalhes, sendo mais adequado para aplicações onde é necessário equilíbrio entre suavização e nitidez.

    2. Variação do Tamanho do Kernel

    O aumento do tamanho do kernel impactou diretamente a intensidade da suavização. Para kernels maiores (como 11x11), todos os filtros apresentaram maior suavização, mas com perda significativa de detalhes. Kernels menores (como 3x3) preservaram mais detalhes, mas foram menos eficazes na redução de ruídos. O filtro bilateral mostrou-se mais estável mesmo com kernels maiores, mantendo as bordas bem definidas.

    3. Resultados em Imagens com Ruído

    Em imagens com ruído do tipo sal-e-pimenta, o filtro mediano foi o mais eficaz, eliminando a maior parte do ruído enquanto preservava as bordas. O filtro Gaussiano também reduziu o ruído, mas com menor eficiência. Já o filtro de média apresentou boa suavização, mas não foi eficiente em casos de ruído intenso. O filtro bilateral manteve um equilíbrio entre remoção de ruídos e preservação de detalhes, mas com maior custo computacional.

    4. Aplicação em Imagens em Tempo Real

    A aplicação dos filtros em imagens capturadas pela webcam permitiu a análise dinâmica dos resultados. Os filtros bilaterais e medianos foram os mais relevantes devido à sua eficácia em lidar com ruídos e preservar bordas. O programa interativo desenvolvido possibilitou uma melhor visualização das diferenças entre os filtros, contribuindo para o entendimento prático.

    Em resumo, a escolha do filtro e do tamanho do kernel deve ser orientada pelo objetivo da aplicação, considerando fatores como a necessidade de suavização, preservação de detalhes e o tipo de ruído presente.

    Conclusão

    Este laboratório proporcionou uma compreensão prática e teórica sobre as diferentes técnicas de filtragem de imagens, destacando suas aplicações, benefícios e limitações. A análise comparativa dos filtros de média, Gaussiano, mediano e bilateral demonstrou que cada método possui características específicas, sendo adequado para diferentes contextos e objetivos.

    Os experimentos realizados evidenciaram a importância de ajustar parâmetros, como o tamanho do kernel, para alcançar os resultados desejados. Filtros como o mediano mostraram-se especialmente eficazes na remoção de ruídos sal-e-pimenta, enquanto o filtro bilateral foi mais versátil ao equilibrar suavização e preservação de bordas.

    A implementação de programas interativos com captura em tempo real reforçou o aprendizado prático, permitindo aos participantes explorar os efeitos dos filtros de forma dinâmica e intuitiva. Esse tipo de abordagem contribui para o desenvolvimento de habilidades aplicáveis em projetos reais de visão computacional.

    Em síntese, este laboratório consolidou os fundamentos do processamento de imagens por meio de filtragem, preparando os participantes para enfrentar desafios mais complexos em aplicações de segmentação, reconhecimento de padrões e análise visual.
